# PROG2001 Assessment 1 - 3D Racing Parkour Greybox Prototype

## Project Overview
This is a 3D racing parkour prototype game developed for PROG2001 Assignment 1. It includes a main menu scene, a player car control system, and basic level design, built with Unity Engine.
All in-game environments and objects are created using Unity’s built-in basic shapes and ProBuilder, with no external assets required.

## Features
- Main menu UI with start/quit buttons
- Basic car movement and camera control
- Greybox level for parkour racing gameplay
- WebGL build for browser-based play

## Project Structure
- `Assets/`: All game scenes, scripts, and resources
- `Packages/`: Unity package management configuration
- `ProjectSettings/`: Project quality and input settings
- `UserSettings/`: Editor layout and preferences
- `Builds/`: WebGL build files (for itch.io deployment)
